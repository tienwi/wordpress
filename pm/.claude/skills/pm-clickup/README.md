# pm-clickup

**Skill del Agent PM de Tienwi** para gestionar proyectos con ClickUp.

## Instalación

```bash
# Desde la raíz del monorepo
bash setup.sh
```

O manualmente:

```bash
cp -r pm/.claude/skills/pm-clickup ~/.claude/skills/
```

## Uso

Una vez instalado, invócalo en Claude Code:

```
/pm-clickup
```

## Requisitos

- ClickUp MCP configurado en `pm/.mcp.json`
- Token de API de ClickUp del workspace Tienwi
